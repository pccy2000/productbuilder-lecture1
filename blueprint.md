# 저녁 메뉴 추천기

## 개요

이것은 오늘 저녁에 무엇을 먹을지 고민하는 사람들을 위해 무작위로 저녁 메뉴를 추천해주는 간단한 웹 애플리케이션입니다.

## 기능

* 다양한 저녁 메뉴 리스트 중 하나를 무작위로 선택하여 추천합니다.
* 사용자 친화적인 인터페이스에 메뉴 이름을 표시합니다.
* 다크 모드/화이트 모드 지원.
* 현대적이고 깔끔한 디자인.

## 프로젝트 구조

* `index.html`: 기본 HTML 파일.
* `style.css`: 스타일링을 위한 CSS 파일.
* `main.js`: 애플리케이션 로직을 위한 JavaScript 파일.

## 변경 이력

* **v1.0**: 로또 번호 생성기
* **v2.0**: 저녁 메뉴 추천기로 변경

## 현재 계획

* `index.html`을 수정하여 로또 관련 요소를 메뉴 추천 관련 요소로 변경합니다.
* `main.js`에서 로또 생성 로직을 제거하고 메뉴 추천 로직을 추가합니다.
* `style.css`를 수정하여 메뉴 텍스트를 표시하기에 적합한 스타일로 변경합니다.
