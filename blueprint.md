# 저녁 메뉴 추천기

## 개요

이것은 오늘 저녁에 무엇을 먹을지 고민하는 사람들을 위해 무작위로 저녁 메뉴를 추천해주는 간단한 웹 애플리케이션입니다.

## 기능

* 다양한 저녁 메뉴 리스트 중 하나를 무작위로 선택하여 추천합니다.
* 사용자 친화적인 인터페이스에 메뉴 이름을 표시합니다.
* 다크 모드/화이트 모드 지원.
* 현대적이고 깔끔한 디자인.
* Google Analytics 연동.

## 프로젝트 구조

* `index.html`: 기본 HTML 파일.
* `style.css`: 스타일링을 위한 CSS 파일.
* `main.js`: 애플리케이션 로직을 위한 JavaScript 파일.

## 변경 이력

* **v1.0**: 로또 번호 생성기
* **v2.0**: 저녁 메뉴 추천기로 변경
* **v2.1**: Google Analytics 추적 코드 추가
