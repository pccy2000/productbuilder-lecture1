# 로또 번호 생성기

## 개요

이것은 무작위 로또 번호를 생성하는 간단한 웹 애플리케이션입니다.

## 기능

* 1과 45 사이에서 6개의 고유한 무작위 번호를 생성합니다.
* 사용자 친화적인 인터페이스에 숫자를 표시합니다.
* 현대적이고 깔끔한 디자인.

## 프로젝트 구조

* `index.html`: 기본 HTML 파일.
* `style.css`: 스타일링을 위한 CSS 파일.
* `main.js`: 애플리케이션 로직을 위한 JavaScript 파일.

## 현재 계획

* 로또 번호 생성기를 위한 기본 HTML 구조를 만듭니다.
* CSS를 사용하여 애플리케이션 스타일을 지정합니다.
* 로또 번호를 생성하고 표시하는 JavaScript 로직을 구현합니다.
